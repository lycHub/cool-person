## 动效流个人网站模板

模板代码完全开源，可用作部署成自己的在线网站(简历)

### 预览

- spa版：[预览](https://yunshixiong.site/spa)
- ssr版：[预览](https://yunshixiong.site/ssr)

### SPA

```shell
pnpm i
pnpm start

# 生产打包
pnpm build

# 预览
pnpm preview
```

### SSR

请切换ssr分支，[并查看说明](https://github.com/lycHub/cool-person/blob/ssr/docs/ssr.md)
